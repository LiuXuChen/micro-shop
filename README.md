## 微服务商城开发平台

### 一、平台介绍

    此平台采用当下最流行的微服务架构进行设计，平台提供最基础的
    开发、权限、单点登录、配置、管理、部署、日志和监控能力，
    开发人员主要利用平台基础能力同时关注业务实现即可，运维人员
    主要利用部署、日志和监控能力进行服务发布、日志查询和系统监控
    保证更快的定位问题和处理问题。
